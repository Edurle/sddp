# 后端接口测试用例 — 项目、迭代

## 1. 项目管理

### 获取项目列表

**接口**: `GET /api/v1/teams/{teamId}/projects`

| 用例编号 | 描述 | 前置条件 | 请求参数 | 预期响应 code | 预期响应 data/说明 |
|----------|------|----------|----------|---------------|-------------------|
| TC-PROJ-001 | 获取成功 | 团队成员，有项目数据 | 无 | 0 | 返回项目列表 |
| TC-PROJ-002 | 按状态筛选active | 同上 | ?status=active | 0 | 仅返回活跃项目 |
| TC-PROJ-003 | 按状态筛选archived | 同上 | ?status=archived | 0 | 仅返回已归档项目 |
| TC-PROJ-004 | end_date取最新迭代截止日期 | 项目有多个迭代 | 无 | 0 | end_date=最新迭代截止日期 |
| TC-PROJ-005 | 无迭代时end_date为空 | 项目无迭代 | 无 | 0 | end_date=null |
| TC-PROJ-006 | 已软删除的项目不显示 | 有已删除项目 | 无 | 0 | 列表不含已删除项目 |
| TC-PROJ-007 | 非团队成员无权查看 | 非成员 | 无 | 40300 | 无权限 |

### 创建项目

**接口**: `POST /api/v1/teams/{teamId}/projects`

| 用例编号 | 描述 | 前置条件 | 请求参数 | 预期响应 code | 预期响应 data/说明 |
|----------|------|----------|----------|---------------|-------------------|
| TC-PROJ-008 | 创建成功 | 有project:create权限 | `{ "name": "项目A", "description": "描述", "start_date": "2026-04-01" }` | 0 | `{ "id": 1 }` |
| TC-PROJ-009 | 名称为空 | 同上 | `{ "name": "" }` | 40001 | 参数校验失败 |
| TC-PROJ-010 | 无权限 | 无project:create权限 | 同TC-PROJ-008 | 40300 | 无权限 |

|----------|------|----------|----------|---------------|-------------------|
| TC-PROJ-011 | 获取成功含统计 | 团队成员 | 路径 id=1 | 0 | 返回详情含statistics |
| TC-PROJ-012 | 项目不存在 | 已登录 | 路径 id=999 | 40400 | 资源不存在 |
| TC-PROJ-013 | 非团队成员 | 非成员 | 路径 id=1 | 40300 | 无权限 |

### 删除项目

**接口**: `DELETE /api/v1/projects/{id}`

| 用例编号 | 描述 | 前置条件 | 请求参数 | 预期响应 code | 预期响应 data/说明 |

| TC-PROJ-021 | 有活跃迭代不可删除 | 项目下有in_progress迭代 | 无 | 40203 | 项目下存在活跃迭代 |
| TC-PROJ-022 | 删除后不在列表 | 已删除 | GET /projects | 0 | 列表不含该项目 |

## 2. 迭代管理

### 获取迭代列表

**接口**: `GET /api/v1/projects/{projectId}/iterations`

| 用例编号 | 描述 | 前置条件 | 请求参数 | 预期响应 code | 预期响应 data/说明 |
|----------|------|----------|----------|---------------|-------------------|
| TC-ITER-001 | 获取成功 | 团队成员，有迭代数据 | 无 | 0 | 迭代列表含requirement_count和task_count |
| TC-ITER-002 | 按状态筛选 | 同上 | ?status=planned | 0 | 仅返回计划中迭代 |
| TC-ITER-003 | 非项目团队成员 | 非成员 | 无 | 40300 | 无权限 |

### 创建迭代

**接口**: `POST /api/v1/projects/{projectId}/iterations`

| 用例编号 | 描述 | 前置条件 | 请求参数 | 预期响应 code | 预期响应 data/说明 |
|----------|------|----------|----------|---------------|-------------------|

| TC-ITER-005 | 截止日期早于开始日期 | 同上 | end_date < start_date | 40001 | 日期不合法 |
| TC-ITER-006 | 名称为空 | 同上 | `{ "name": "" }` | 40001 | 参数校验失败 |
| TC-ITER-007 | 无权限 | 无iteration:create权限 | 同TC-ITER-004 | 40300 | 无权限 |

| TC-ITER-014 | 已启动不可重复启动 | status=in_progress | 无 | 40204 | 迭代状态不允许 |
| TC-ITER-015 | 已完成不可启动 | status=completed | 无 | 40204 | 迭代状态不允许 |
| TC-ITER-016 | 无权限 | 无iteration:start权限 | 无 | 40300 | 无权限 |

### 完成迭代

**接口**: `POST /api/v1/iterations/{id}/complete`

| 用例编号 | 描述 | 前置条件 | 请求参数 | 预期响应 code | 预期响应 data/说明 |
|----------|------|----------|----------|---------------|-------------------|
| TC-ITER-017 | 完成成功 | 所有需求已拆分任务+所有任务completed+测试全通过 | 无 | 0 | 迭代已完成 |
| TC-ITER-018 | 存在未完成的需求 | 有需求未拆分任务 | 无 | 40200 | 存在未完成的需求 |
| TC-ITER-019 | 存在未完成的任务 | 有任务status!=completed | 无 | 40201 | 存在未完成的任务 |
| TC-ITER-020 | 存在未通过的测试 | 最近测试有失败用例 | 无 | 40202 | 存在未通过的测试 |
| TC-ITER-021 | 非进行中状态不可完成 | status=planned | 无 | 40204 | 迭代状态不允许 |
